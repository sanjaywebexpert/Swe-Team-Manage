== Description ==

SWE Team Management For WordPress is plugin which help you to manage and display your team members on carousel slider with many layouts. This will create a separate admin section for Team in admin area where to manage team/staff/employee on your sites.
This plugin is very ease of use by providing different shortcodes for different layouts of team slider and team grid view. This is responsive layout of Team design suitable for all devices.

Separate Setting pages for color, fonts and other checkboxes for on off to show / hide social, title, description in team layouts. Easy to use and manage.

\[Compatible with all PHP version.\]

This is completely free version, all the pro features as in this free version.

= Available fields =
* Name 
* Position/ Job Title/ Designation
* Short Description
* Photo
* Social links (FB, Twitter…)

= Available settings =
* Corousel Loop 
* Show/hide pagination
* Navigation Position
* Autoplay
* Number of Team to show per line in Desptop, Tablet and mobile
* Color
* Show/hide Title, Description, social...

== Installation ==

1. Unzip and upload plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Swe Team admin menu and add teams and use the layouts as you desire with given shortcodes
4. Just add shortcodes to pages and posts to display in front-end.
